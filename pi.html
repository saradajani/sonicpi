What is Sonic Pi?

Sonic Pi is an open-source programming environment, designed for creating new sounds with code in a live coding environment. Sonic Pi is a live coding environment based on Ruby. It was developed by Dr Sam Aaron at the University of Cambridge. He uses the software to perform live with his band.

What was the project?

Technology is a fantastic tool for creative expression. During this project students learned about computing constructs and computational thinking through exploration and discovery as they generated digital music. Using a scripting language based on Ruby, students created sounds and ultimately composed and performed a song for the CS Pathway class. This composition would abstract features of music using code. During this project students demonstrated mastery of ubiquitous programming ideas including loops, variables, arrays, randomness, control flow, functions, and parameterized functions.
